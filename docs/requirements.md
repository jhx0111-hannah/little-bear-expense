# 小熊记账 — 项目需求文档

## 项目背景

用户"小熊"是一名在欧洲留学的中国学生。日常同时使用人民币（CNY）和欧元（EUR），需要一个支持多币种、操作简单、风格可爱的记账App。

## 目标用户

- 在欧洲留学的中国学生
- 日常使用人民币 + 欧元双币种
- 非技术背景，需要简洁易用的界面

## 核心功能需求

### F1. 用户认证
- 邮箱 + 密码登录/注册（自动识别新用户）
- 无需邮箱验证，简化注册流程
- 支持退出登录

### F2. 多币种记账
- 支持人民币（CNY）和欧元（EUR）
- 每笔记账记录：金额、币种、分类、日期、备注、扣款账户
- 支持支出和收入两种类型
- **三种记账方式**：
  - 手动填写表单
  - 截图上传 → AI自动识别金额/商户/分类并填表
  - 语音输入 → 说话后AI自动解析金额/分类/时间，生成账单草稿

### F3. 账户管理
- 添加个人银行/支付账户（ING、Revolut、支付宝、微信、招商银行等）
- 记录每个账户的币种和当前余额
- 记账时选择扣款账户，余额自动扣减
- 支持编辑余额和删除账户

### F4. 资产总览
- 展示所有账户余额
- 按实时汇率换算总资产
- 显示主要货币总额 + 次要货币约合额

### F5. 统计分析
- 日/周/月/年维度切换
- 饼状图：各分类支出占比
- 柱状图：支出 vs 收入对比
- 月度每日明细表（基金收益风格）
- 月度累计汇总

### F6. 流水明细
- 按月份查看所有交易记录
- 支持搜索和日期范围筛选
- 每笔显示：分类、描述、金额、币种、时间、扣款账户

### F7. 设置
- 修改昵称
- 设置默认货币
- 设置月预算
- 自定义分类管理
- 数据导出（CSV）

### F8. PWA支持
- 可添加到手机主屏幕
- 离线模式下可查看已缓存数据
- 启动画面和应用图标

### F9. 语音记账
- 点击麦克风按钮开始说话（如"今天中午在麦当劳花了35元"）
- 使用浏览器内置语音识别（Web Speech API），支持中文识别
- 识别出的文字发送给AI，解析出：金额、币种、分类、商户、日期
- 自动生成账单草稿，预填表单
- 用户确认/修改后保存
- 支持识别自然语言中的时间表达（"昨天"、"上周五"、"今天中午"等）
- 如果信息不完整（如只说"花了50块"没说什么分类），AI自动推断或留空让用户补充

## 非功能需求

- 全中文界面
- 移动端优先设计（适配375px宽屏幕）
- 莫兰迪色系（低饱和度柔和色调）
- 小熊+可爱小动物主题元素
- 数据云端存储（Supabase），支持多设备同步
- 响应速度：页面切换 < 300ms，AI识别 < 5s
