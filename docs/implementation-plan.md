# 小熊记账 — 实施步骤

## 开发原则

1. **小步推进**：每个阶段完成后验证，确保稳定再进入下一阶段
2. **先骨架后血肉**：先让页面能跑通，再加细节和动画
3. **移动端优先**：每完成一个页面就在手机上验证效果
4. **及时记录**：每完成一个阶段更新 dev-log
