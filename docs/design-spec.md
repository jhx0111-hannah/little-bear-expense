# 小熊记账 — 设计规范

## 一、品牌标识

- **名称**：小熊记账
- **英文**：Little Bear Expense Tracker
- **吉祥物**：小熊（出现在Logo、空状态、加载动画中）

## 二、莫兰迪色系（Morandi Color Palette）

莫兰迪色系特点：低饱和度、柔和、沉稳、温暖，不刺眼。

### 色板

```css
:root {
  /* === 主色调 === */
  --color-sage:        #b5c4b1;  /* 鼠尾草绿 — 主按钮、强调 */
  --color-dusty-rose:  #c4a8a8;  /* 灰玫瑰 — 支出相关 */
  --color-lavender:    #b5a9b0;  /* 灰紫 — 中性元素 */
  --color-powder-blue: #a8b5c4;  /* 粉蓝 — 链接、信息 */
  --color-warm-beige:  #d4c9b8;  /* 暖米色 — 背景 */
  --color-muted-teal:  #a3b5a6;  /* 茶绿 — 收入相关 */

  /* === 背景 === */
  --bg-primary:        #faf7f2;  /* 暖白（主背景） */
  --bg-secondary:      #f5f0eb;  /* 稍暖（卡片背景） */
  --bg-card:           #ffffff;  /* 纯白（浮层卡片） */

  /* === 文字 === */
  --text-primary:      #4a4444;  /* 深灰棕 */
  --text-secondary:    #8c8484;  /* 中等灰 */
  --text-hint:         #b8b0b0;  /* 浅灰提示 */

  /* === 语义色 === */
  --color-income:      #a3b5a6;  /* 收入绿 */
  --color-expense:     #c4a8a8;  /* 支出玫瑰 */
  --color-danger:      #c49a9a;  /* 警示红 */
  --color-warning:     #d4c4a8;  /* 提醒黄 */

  /* === 小熊主题 === */
  --color-bear-brown:  #b5a08a;  /* 熊棕 */
  --color-bear-pink:   #d4b8b8;  /* 熊粉（腮红） */
}
```

### 使用原则

- 主按钮用鼠尾草绿
- 支出金额用灰玫瑰
- 收入金额用茶绿
- 背景用暖米白，卡片用纯白
- 不要使用高饱和度的纯色

## 六、小熊元素使用场景

| 场景 | 小熊状态 | 说明 |

## 七、图标

- 分类图标使用Emoji（🍜🚌🛒🎮📚🏥🏠💰💵🧧📥）
- 账户类型图标也使用Emoji（🏦📱💳💵）

## 八、动画规范

- 页面过渡：淡入+轻微上移，250ms
- 按钮点击：缩放到97%，150ms
- 小熊呼吸动画：慢速缩放，2s循环
- 数字跳动：轻微弹跳，300ms
- AI识别完成：表单项逐一亮起高亮，各延迟100ms
- **语音按钮监听中**：麦克风图标脉冲缩放（1s循环），外圈波纹扩散
- **语音波形**：录音时显示音频波形条（5根不同高度的莫兰迪色条，随音量变化）

## 九、移动端适配

- 设计基准宽度：375px（iPhone 6/7/8）
- 最大内容宽度：480px，超出居中留白
- 底部导航高度：56px + safe-area-inset-bottom
- 顶部状态栏预留：env(safe-area-inset-top)
- 所有可点击区域最小 44x44px（符合iOS规范）

## 十、语音记账UI规范

### 语音按钮（记账页麦克风按钮）

```
┌──────────────────┐
│                  │
│    🎤 按住说话    │  ← 莫兰迪圆形按钮（鼠尾草绿），80px
│                  │
└──────────────────┘
```

- **位置**：记账页顶部，金额输入框上方
- **默认态**：鼠尾草绿圆形按钮，中间白色麦克风图标
- **按住态**：按钮放大至110%，外圈出现波纹扩散动画，小熊竖起耳朵
- **识别中**：按钮变为旋转加载动画，文字显示"小熊正在听..."
- **完成态**：按钮弹回，显示绿色勾 + "识别完成"，0.5s后自动填入表单
- **失败态**：按钮变为灰玫瑰色，显示"没听清楚，再试一次？"

### 语音波形指示器

录音时在麦克风按钮上方显示5根波形条：
- 使用莫兰迪色系5个层级（深→浅）
- 高度随实际音量动态变化
- 无声时最小高度4px，最大音量时32px
- 给用户视觉反馈"正在收音"

### 语音识别结果预览

识别完成后，在记账表单上方短暂显示一个解析结果卡片：
```
┌──────────────────────────────┐
│  🤖 小熊听懂了：              │
│  金额：¥35.00                │
│  分类：🍜 餐饮               │
│  商户：麦当劳                 │
│  时间：今天 12:30            │
│  ──────────────────────────  │
│  [ ✓ 确认 ]  [ ✗ 重新说 ]   │
└──────────────────────────────┘
```

用户点确认后自动填入表单，点重新说则清除并重新录音。

### 兼容性处理
- iOS Safari 14.5+ 支持 Web Speech API
- Android Chrome 支持
- 如果浏览器不支持语音识别，隐藏麦克风按钮（不显示比显示后报错好）
